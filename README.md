# Single Paramter Duplicator

*Created for JetBrains intership projects 2025-2026*

Small program that takes a C# source file as input, and duplicates the parameter of every method with only one parameter. It suggests a new name, but also let's the user choose their own.

## Installation

Clone the repository and navigate to the folder

```zsh
git clone https://github.com/markovolimango/StructuralDuplication.git
cd StructuralDuplication
```


## Usage

### Running the tests

```zsh
dotnet test
```

### Refactoring a file

1. Run the project

```zsh
dotnet run --project StructuralDuplication
```

2. Follow the instructions in the terminal (enter the path to your file and choose new parameter names) and your file is reformated!
